Resources

A resource is an entity that is managed by #{msgs.title}. For example, a name element, a name category, a name request etc. The #{msgs.title} API provides access to the resources or entities that it manages. These entities and their attributes are described below. Note that the names of attributes are important as they are used in the various representations of these resources (see Resource Representations Section below).
