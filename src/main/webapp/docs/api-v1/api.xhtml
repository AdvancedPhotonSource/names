API

The following table lists the interfaces to manage various Hour Log resources. It lists the resource name, its URI, the HTTP method, its parameters, and the response from the API to the method.

Name Method Resource ID Description Parameters Examples Name Element GET /name List of Name Elements. By default, only approved name elements are returned unless 'all' parameter is true, in which case all names including 'in-process', 'deleted', 'rejected' etc are returned.
  • category: Events for Name Elements of given category
  • all: (true or false) Include all name elements
  • #{restRoot}/name
  • #{restRoot}/name?category=system
  • #{restRoot}/name?all=true
 Category GET /category List of categories of name elements.
  • #{restRoot}/category
 Event GET /event List of all events of all naming elements
  • category: Events for Name Elements of given category
  • deleted: (true or false) Include deleted name elements
  • #{restRoot}/event
  • #{restRoot}/event?category=system
 Release GET /release List of all releases
  • #{restRoot}/release
